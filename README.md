## 🔐 Steganography Image Encoder & Decoder Web Application

**Project Description :**
Developed a browser-based steganography application that enables users to securely **hide and extract secret messages inside digital images using the Least Significant Bit (LSB) encoding technique**. The system provides an interactive web interface where users can upload images, embed confidential text data, and decode hidden information without noticeable visual distortion in the image. Implemented client-side image processing logic, binary data manipulation, and dynamic UI components to demonstrate practical concepts in information security and data hiding techniques. This project strengthened understanding of cybersecurity fundamentals, image encoding algorithms, and frontend-based file handling workflows.

**Technologies Used:**

* HTML
* CSS
* JavaScript
* Image Processing (LSB Steganography Algorithm)
* Browser File Handling APIs
